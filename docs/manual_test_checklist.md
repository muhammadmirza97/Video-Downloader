# Manual Test Checklist

Use this checklist before sharing the app with a beginner.

## Setup

- Double-click `launch_app.bat`.
- Confirm the app opens without showing a Python error.
- Confirm a `downloads` folder is created only when needed.

## Download Flow

- Paste a normal YouTube video URL.
- Keep the default folder and click `Download MP4`.
- Confirm the progress bar moves or the status text changes.
- Confirm the final video appears in the selected folder.
- Click `Open Folder` and confirm File Explorer opens the same folder.

## Folder Selection

- Click `Browse...` and select a different folder.
- Start a download and confirm the file saves there.
- Click `Browse...`, cancel the dialog, and confirm the old folder stays selected.

## Error Handling

- Leave the URL blank and click `Download MP4`.
- Paste a non-YouTube URL and click `Download MP4`.
- Disconnect internet temporarily and try a download.
- Try a private, removed, or unavailable video.
- Confirm the app shows a clear message and does not freeze.

## Repeated Use

- Download into a folder that already has a video with the same name.
- Start the app again after closing it.
- Confirm the app is still understandable without reading code.

## Safety Review

- Confirm the app reminds users to download only videos they own, have permission to use, or are allowed to download.
- Do not add features for bypassing protections, bulk scraping, login-cookie workarounds, or redistribution.
